Usage
=====

.. _installation:

Installation
------------

To use the POPGym environments, install it using pip:

.. code-block:: console
   $ git clone https://github.com/smorad/popgym
   $ cd popgym
   $ pip install .

To install the baselines and dependencies, first install ray then
install POPGym

.. code-block:: console
   $ pip install ray[rllib]
   $ git clone https://github.com/smorad/popgym
   $ cd popgym
   $ pip install ".[baselines]"
